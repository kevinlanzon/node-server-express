Node Server with Express
==========

A simple tested Node server using Express.js.

###Technologies used

- Node.js
- Express.js
- Mocha
- Chai
- Grunt


###How to run

```sh
$ git clone git@github.com:kevinlanzon/node-server-express.git
$ npm install
$ selenium-standalone start
$ npm start
$ grunt
Visit http://localhost:3000
```
